Data

Currently, the Virtual Valdivia database includes ceramic artifacts derived from excavations at the site of Buen Suceso, and focuses on Phases VI and VII of the Late Valdivia period. Buen Suceso is located in the Manglaralto Valley of Santa Elena Province, Ecuador, and was occupied between 2100 and 1800 BC.

It is my hope that sharing my data through the Virtual Valdivia database can serve as an example to encourage other scholars to share their data. In the future, Virtual Valdivia should develop into a ceramic repository for multiple Valdivia sites from Ecuador, facilitating analysis across sites. Additional data will be added from grey literature, such as unpublished theses and dissertations and site reports as author permission is obtained. If you are interested in contributing your data please contact the database editor, Dr. Sarah M. Rowe.

Using the Database

The Virtual Valdivia database is hosted by Open Context. Each ceramic piece is associated with numerous variables. Descriptions of each of these variables can be viewed on Open Context, or you can download the complete set of variable documentation as a PDF document by clicking here. Visit the complete Virtual Valdivia data set on Open Context to browse through artifacts and read the accompanying documentation.

You can use the search box below to access a subset of the results (by searching for one of the data set variables), or you can download the complete data set as a .csv file by clicking here (coming soon).

If you plan to use the data, please follow citation guidelines and give credit to the database author.
